# Login Form Streamlit App

## 📌 Overview

This project is a simple **Login Form Web Application** built using **Streamlit** and **Python**.
It allows users to log in using credentials stored in a database (MySQL).

---

## 🚀 Features

* User login authentication
* Simple and interactive UI using Streamlit
* MySQL database connectivity
* Error handling for invalid login
* Easy to run and modify

---

## 🛠️ Technologies Used

* Python
* Streamlit
* MySQL
* mysql-connector-python

---

## 📂 Project Structure

```
Login_form_streamlitAPP/
│── login.py           
│── task.py            
│── requirements.txt    
│── README.md           
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone <your-repo-link>
cd Login_form_streamlitAPP
```

2. Create virtual environment:

```
python -m venv venv
venv\Scripts\activate   
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the App

```
streamlit run login.py
```

---

## 📸 Output

* Login page will open in browser
* Enter username and password
* Displays success or error message
